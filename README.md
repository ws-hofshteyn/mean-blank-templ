**MEAN stack blank templ**

If you have already installed all global dependencies such as Gulp and etc.

Step 1
```
git clone https://github.com/ws-hofshteyn/mean-blank-templ.git
```
Step 2
```
cd mean-blank-templ
```

Step 3
```
npm install
```
Step 4
```
bower install
```
Step 5
```
gulp  
```

Step 6
Go to http://localhost:3000
